## 🌟 我們的核心訓練

以下為初學者與想重拾運動習慣的你，量身打造最合適的健身規劃：

### 🏋️‍♂️ 肌力訓練 (Strength Training)
透過系統化且循序漸進的重量訓練，有效刺激肌肉生長、增強力量。不僅能改善身體組成、降低體脂肪，更能提升基礎代謝率，讓你連躺著都在燃燒熱量！同時還能強化關節穩定度，讓生活中的搬抬、走樓梯都變得輕輕鬆鬆。

### 🎓 技術指導 (Skill Guidance)
完全沒有經驗的健身新手免操心！教練會一步步教授正確的動作姿勢、精準的呼吸方法與核心出力觀念，帶你遠離盲目摸索，徹底防範錯誤動作帶來的受傷風險。

### 📈 週期訓練 (Periodization)
導入科學化的訓練週期規劃，將訓練分為不同階段。依照你的個人狀態與目標調整強度與內容，確保每個階段都能持續進步，有效打破訓練停滯期，讓你的每一分汗水都能看見實質改變！

## 🛠️ 上課小叮嚀

{% note modern 'fas fa-bullseye' %}
**100% 客製化個人需求**
每個人的身體狀況、作息都不同，不論你的目標是科學減脂、雕塑線條還是體態調整，教練都會依據你的需求客製化專屬的黃金課表。
{% endnote %}

{% note modern 'fas fa-clock' %}
**彈性授課時間**
每堂課程完整 **60 分鐘**。預約時段高自由度（**10:00 ~ 20:00**），下班課後都能彈性安排，不影響你的日常生活。
{% endnote %}

{% note modern 'fas fa-shoe-prints' %}
**上課準備與流程**
請穿著舒適的運動服裝與室內運動鞋（嚴禁涼鞋或拖鞋唷！）。建議提早 10 分鐘到場自主暖身活化，且切勿空腹運動，才能保持最佳的訓練狀態。
{% endnote %}
